# Aloyadua-Landing Page

Um card pessoal minimalista e estilizado criado com **HTML**, **CSS** e **JavaScript**.

O projeto funciona como uma página de apresentação/identidade visual, reunindo nome, imagem, frase, links sociais e uma estética misteriosa com elementos visuais personalizados.

## 🌐 Preview

Acesse o projeto publicado:

https://aloyadua.github.io/ACard/

## ✨ Sobre

**ACard** é uma landing page simples, direta e visualmente marcante.  
A proposta é servir como um cartão digital pessoal, reunindo links importantes em uma interface única.

A página apresenta:

- Nome/identidade principal
- Imagem ou avatar personalizado
- Frase de impacto
- Links para redes sociais
- Estilo visual próprio
- Estrutura leve e fácil de editar
